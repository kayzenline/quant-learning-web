# 量化交易学习中心 🚀

从零到一系统学习量化交易的现代化Web平台

![Hero Banner](assets/images/hero-bg.png)

## 📖 项目简介

这是一个精心设计的量化交易学习网站，为初学者提供从基础到进阶的完整学习路径。网站采用现代化设计语言，重点突出**AI驱动的因子挖掘**和**社交媒体情绪分析**两大创新方向。

## ✨ 核心特色

### 🤖 AI驱动创新

- **自动因子挖掘**: 使用机器学习自动发现和迭代优化交易因子
- **情绪指标开发**: 通过爬虫技术抓取X(Twitter)和主流媒体，构建实时情绪指标

### 🎨 现代化设计

- 深色主题配合渐变色和玻璃态效果
- 流畅的滚动动画和交互
- 完美的响应式设计
- 专业的视觉层次

### 📚 系统化学习路径

1. **基础入门** - 量化交易概念与市场结构
2. **数学与统计** - 概率论、时间序列分析
3. **Python编程** - 数据处理、API调用
4. **AI因子挖掘** - 机器学习自动发现因子
5. **情绪指标开发** - 爬虫与NLP分析
6. **策略开发** - 回测框架与性能评估
7. **风险管理** - 仓位管理、VaR计算
8. **实盘部署** - 从模拟到实盘落地

## 🚀 快速开始

### 本地运行

```bash
# 克隆项目（如果使用Git）
# cd 到项目目录
cd quant-learning-website

# 启动本地服务器（Python 3）
python3 -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000

# 在浏览器中访问
open http://localhost:8000
```

### 直接打开

也可以直接在浏览器中打开 `index.html` 文件，但建议使用HTTP服务器以获得最佳体验。
