# 部署到 GitHub Pages

本文档记录了将量化交易学习网站部署到 GitHub Pages 的步骤。

## 📋 部署步骤

### 1. 初始化 Git 仓库

- [ ] 初始化本地 Git 仓库
- [ ] 添加 .gitignore 文件
- [ ] 提交初始代码

### 2. 创建 GitHub 仓库

- [ ] 在 GitHub 创建新仓库
- [ ] 连接本地仓库到 GitHub
- [ ] 推送代码到 GitHub

### 3. 配置 GitHub Pages

- [ ] 启用 GitHub Pages
- [ ] 配置部署分支
- [ ] 验证网站上线

## 🔗 最终网址

部署完成后，网站将在以下地址可访问：
`https://您的GitHub用户名.github.io/quant-learning-website`

## 📝 注意事项

- 确保仓库设置为 Public（公开）以使用免费的 GitHub Pages
- 首次部署可能需要几分钟时间
- 每次推送新代码都会自动重新部署
